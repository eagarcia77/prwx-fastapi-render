# Fuentes de datos recomendadas

## NOAA/NCEI CDO y GHCN-Daily

Uso: datos históricos de estaciones, temperatura, precipitación, viento y resúmenes diarios.

## NWS API

Uso: pronósticos oficiales por punto/cuadrícula, observaciones y alertas activas.

## MRMS QPE

Uso: lluvia estimada por radar a alta resolución. Ideal para entrenar lluvia observada espacialmente.

## USGS NWIS

Uso: caudal, nivel de ríos, precipitación en estaciones hidrológicas y respuesta de cuencas.

## NASA POWER

Uso: variables meteorológicas y solares para lugares donde faltan estaciones.

## ERA5

Uso: reanálisis histórico para variables atmosféricas de largo plazo.
