# Accesibilidad WAVE/WebAIM en PR-WX v0.9

La versión v0.9 fue rediseñada para ser más fácil de entender y más compatible con una revisión WAVE.

## Mejoras incorporadas

- Alto contraste entre texto y fondo.
- Encabezados claros y jerárquicos.
- Botones y controles con etiquetas visibles.
- Enfoque visible para navegación por teclado.
- Opción para reducir animación.
- El mapa animado tiene alternativa textual en tabla.
- Las alertas usan texto, severidad y acción recomendada; no dependen solo del color.
- Pueblos prioritarios aparecen en tarjetas y tablas.

## Validación recomendada

1. Ejecutar el dashboard.
2. Abrir `http://localhost:8501`.
3. Usar la extensión WAVE del navegador.
4. Revisar contrastes, etiquetas y estructura generada por Streamlit.

Streamlit genera parte del HTML dinámicamente, por lo que la validación final debe hacerse desde el navegador.
