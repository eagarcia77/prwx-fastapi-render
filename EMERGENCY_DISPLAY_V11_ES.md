# PR-WX v1.1 Emergency Display

Esta versión reorganiza la experiencia como un **centro de mando** más claro, útil para proyectarse en pantalla grande,
kiosco, sala de operaciones o laboratorio educativo.

## Mejoras principales

- Vista inicial tipo **semáforo** con prioridad clara.
- Tarjetas grandes para **Juana Díaz, Ponce, San Juan y San Germán**.
- Mensajes de **qué hacer ahora** en lenguaje llano.
- Enfoque de accesibilidad: alto contraste, títulos claros y alternativa textual del mapa.
- Preparado para añadir notificaciones sonoras y web push en una próxima iteración.

## Objetivo

Reducir la carga cognitiva y facilitar que una persona pueda responder rápido:
1. Ver el riesgo principal.
2. Identificar los pueblos prioritarios.
3. Leer la acción sugerida.
4. Revisar temperatura, lluvia, viento y alertas.
